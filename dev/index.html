Utility.iNext — Method
iNext(i::Integer, n::Integer) -> Integer

Given n elements in an array, return the next i+1 element after i, wrapping around 0 if necessary.

Utility.iNext(1,5) == 2
Utility.iNext(5,5) == 1
Utility.iNext(6,2) == 1
Utility.iNext(-6,5) == 0

iWrap(i::Integer, n::Integer) -> Integer

Utility.stringTable — Function

stringTable(strings::Vector{String}, lengthEach::Integer=3, separator::String="")::String Regularizes the strings vector by truncating each string to lengthEach then concatenating into one string, separated by separator.

Utility.stringTable(["abc","defghi","jk","lmnopq"], 4, "|") == " abc|defg|  jk|lmno"
Utility.stringTable(["abcdef","ghi","jkl","mnopq"], 4, "|") == "abcd| ghi| jkl|mnop"
